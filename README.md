PrimeEstate is a full-stack application for property listing, searching, and agent management, built on a modern tech stack (e.g., React, Node.js, MongoDB).

To run the app, install Node.js, clone the repository, install dependencies in the server and client folders, set environment variables, and start both the server and client processes.
